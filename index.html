git command

1. git init

initialize the git to the project directory

2. git status

to check the untracked and tracked file status before commiting or adding to git

3. git add filename

example - git add index.html

to add single file to git cache if changes is done

4. git add .

to add all file to git cache if changes is done

5.git rm --cached filename

to remove added file fro cached

git restore filename

to remove all new change if the changes are not added to git cache

7.git commit -m "msg"

eg- git commit -m "project initialized and added index.html"

to add the git file here -m is used to add message for command

8.git remote add origin https//ssh repolink

eg git remote add origin git@github.com:rabin1783/weather-apps.git

to add the remote server respository to local server

9.git show

to check the file content

10.git push -u origin branchname

eg git push -u origin master

to push our changes to remote server

if we want to push in master then branch name = master
